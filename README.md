Employee Management System (EMS)

A modern Employee Management System built using HTML, CSS, and JavaScript that helps manage employee data, attendance, and leave records in a clean and responsive dashboard interface.

This project provides a glassmorphism-based UI, a responsive sidebar, and an interactive dashboard to manage workforce information efficiently.

🚀 Features

📊 Dashboard Overview

Total Employees

Active Employees

Employees on Leave

Quick statistics cards

👨‍💼 Employee Management

Add new employees

Upload employee profile image

View employee details in cards

Edit employee information

Delete employees

Prevent duplicate employees

🔍 Search Functionality

Search employees instantly using the search bar.

📅 Attendance Management

Track employee attendance

Mark present/absent

📝 Leave Management

Apply leave

View leave records

📱 Responsive Design

Mobile friendly layout

Collapsible sidebar for small screens

🎨 Modern UI

Glassmorphism interface

Animated sidebar

Clean dashboard cards

🛠️ Technologies Used

HTML5 – Structure of the application

CSS3 – Styling and responsive design

JavaScript (Vanilla JS) – Application logic and interactivity

Local Storage – Store employee data in the browser
